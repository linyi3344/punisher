# punisher
a simple js web crawler

## how to use 
```
+1. npm i
```

+2. modify `index.js`
change start pge
```
//start page
_config._schedules.push('http://bj.fang.lianjia.com/loupan');
```
```
+3. npm run punisher
```
+4. then you will see result in `result.txt` (meanwhile you can change output file in `index.js` )
